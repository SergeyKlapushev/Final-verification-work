# ИТОГОВАЯ КОНТРАЛЬНАЯ РАБОТА 

Пишем программу, которая из имеющегося массива строк формирует 
массив из строк, длинна которого меньше либо равна 3 символа.

## Пример:

    ['Hello' '2' 'world' ':-)' ] -> ['2', ':-)']
    ['1234' '1567', '-2', 'computer science'] -> ['-2']
    ['Russia', 'Denmark', 'Kazan'] -> []
